# Bowl of Ramen


## Getting started

```bash
git clone https://github.com/averageeucplayer/bowl-of-ramen
cd bowl-of-ramen
cargo build bowl-of-ramen
cargo tauri dev
```

## Debug

App

```bash
cd app
cargo tauri dev
```

Web

```bash
npx tailwindcss -i ./styles/main.css -o ./styles/output.css --watch
trunk build
trunk serve
```
